# UploadRanger 更新日志

## v1.1.1 (2026-04-06)

### Bug 修复

- **标签页关闭崩溃**: 修复 Repeater/Intruder 关闭最右侧标签页导致程序崩溃/无响应的问题
- **程序退出卡顿**: 优化关闭流程，确保后台线程正确停止
- **扫描结果到 Repeater**: 修复请求数据包重复请求行的问题
- **日志简化**: 移除不必要的调试输出
- **请求历史颜色渲染**: 修复成功响应（2xx）没有绿色背景的问题
- **请求历史ID**: 确保使用唯一的请求ID而非行号
- **右键发送到Repeater/Intruder**: 修复请求格式，确保请求数据完整传递
- **Polyglot图片马生成**: 修复PHP标签重复添加问题，用户输入已包含`<?php`标签时不再重复添加

### 界面调整

- 主窗口尺寸调整为 1315x875

### 核心引擎升级

- **RawHTTPClient**: 原始Socket HTTP客户端，字节级HTTP控制
  - 精确控制 multipart/form-data boundary
  - 支持 filename 编码绕过（null_byte, double_ext, case_flip 等）
  - 支持自定义 boundary 注入
  - 支持代理和 SSL
- **SmartAnalyzer**: 智能响应分析器，三级判定机制
  - Level 1: 快速排除明显失败
  - Level 2: 综合打分（0.0-1.0置信度）
  - Level 3: 执行验证
  - 自动 WAF 检测
  - 隐藏指示器检测
- **Scanner 集成新核心引擎**:
  - 支持切换使用 RawHTTPClient 或传统 HTTPClient
  - Payload 数量提升至 1200
  - 更准确的响应判定

### GUI 与工程

- **Repeater / Intruder 多标签页**: 支持多标签页管理，可新建/关闭/重命名标签，标签页支持拖拽排序
- **标签页交互优化**: 标签栏使用紧凑样式（固定宽度 80-150px），双击标签重命名对话框增大至 400x150 像素
- **响应视图增强**: 隐藏无用的 Image 标签；请求/响应界面响应区域使用 ResponseViewerWidget，支持 Raw/Pretty/Render/Hex 多视图
- **扫描控制改进**: 优化停止扫描逻辑，立即响应避免 UI 冻结；进度更新频率增强，显示更平滑
- **异步扫描对齐同步能力**: 可选字节级 Raw 上传、环境指纹预检与 Payload 过滤/排序；流量面板记录 Raw 请求摘要
- **版本统一**: `config.VERSION` / 关于页 / 标题栏一致为 **1.1.0**；测试脚本横幅同步
- **检查更新**: 后台线程拉取 GitHub Release，经 `QTimer.singleShot` 回主线程弹窗，避免界面卡死
- **扫描 Payload 说明**: 界面「Payload 数量」为上限，实际请求数 ≤ min(上限, 内置词库条数)；开启指纹后还可能更少
- **新手向导**: 快速填写目标 URL、字段名、上传目录并跳转扫描页
- **插件占位**: `plugins/schema.json` 预留社区 Payload 插件描述格式

### 靶场修复

- **Level 6/10 Magic Bytes Bug**: 修复转义字符串写法（`b'\\xff'` → `b'\xff'`）
- **Level 9 路径穿越**: 补充绝对路径过滤（`/`, `\`, `:`）
- **Level 8 双扩展名**: 修正检查逻辑（原检查任意位置，现只检查末尾）
- **安全性**: debug模式改为默认关闭，仅监听本地

## v1.0.5 (2026-03-24)

### 判定逻辑升级

- **证据分层 + 冲突裁决**: 异步响应分析从"单关键词判定"升级为"结构化 JSON + 文本关键词 + 状态码 + 路径"综合裁决
- **结构化成功证据优先**: 支持识别 `success=true`、`files` 非空、`message` 成功语义
- **判定可解释**: 新增 `decision_reasons`（判定原因）与 `confidence_level`（置信等级）

### 上传验证增强

- **服务端重命名适配**: 从响应 JSON 提取服务端保存名
- **多候选验证**: 支持候选文件名逐个验证
- **触发条件优化**: 成功概率达到阈值时尝试二次验证

### 界面与版本

- 扫描结果表新增判定说明提示
- 项目版本更新为 **v1.0.5**
